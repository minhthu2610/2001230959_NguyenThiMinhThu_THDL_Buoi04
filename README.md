# DEEP LEARNING - THỰC HÀNH TUẦN 4

## Giới thiệu

Repository này bao gồm các bài thực hành môn Deep Learning nhằm làm quen với quy trình xây dựng, huấn luyện và đánh giá các mô hình học sâu sử dụng TensorFlow/Keras.

Các bài thực hành tập trung vào bài toán phân loại ảnh bằng mạng nơ-ron tích chập (CNN - Convolutional Neural Network), từ các bộ dữ liệu cơ bản đến các bộ dữ liệu thực tế hơn.

## Nội dung thực hiện

Trong tuần học này, các nội dung chính đã thực hiện gồm:

* Tiền xử lý dữ liệu ảnh.
* Chuẩn hóa dữ liệu đầu vào.
* Chia tập huấn luyện và kiểm thử.
* Xây dựng mô hình CNN.
* Huấn luyện mô hình bằng TensorFlow/Keras.
* Đánh giá hiệu năng mô hình thông qua Accuracy và Loss.
* Thực hiện dự đoán trên dữ liệu mới.
* Trực quan hóa kết quả huấn luyện bằng biểu đồ.

## Công nghệ sử dụng

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Google Colab

## Kết quả đạt được

* Hiểu được quy trình xây dựng mô hình Deep Learning cho bài toán phân loại ảnh.
* Làm quen với kiến trúc mạng CNN và các thành phần cơ bản như Convolution, Pooling, Flatten và Dense.
* Thực hiện thành công việc huấn luyện và đánh giá mô hình trên các bộ dữ liệu khác nhau.
* Nắm được các kỹ thuật tiền xử lý và tăng cường dữ liệu (Data Augmentation) nhằm cải thiện hiệu suất mô hình.

## Kết luận

Thông qua các bài thực hành, đã củng cố kiến thức về Deep Learning và mạng CNN, đồng thời rèn luyện kỹ năng triển khai mô hình học sâu bằng TensorFlow/Keras trên môi trường Google Colab. Đây là nền tảng quan trọng cho các bài toán nhận dạng ảnh và thị giác máy tính trong thực tế.
